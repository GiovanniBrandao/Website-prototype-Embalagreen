Aqui estão os arquivos do site, feito em HTML, CSS e JavaScript, que 
conta com 3 páginas (Inicial, Parcerias e Contato) representadas por
index, pag02 e pag04. Os arquivos em CSS também 
estão denominados para cada página.

Além disso, contém a pasta de imagens utilizadas e fontes
para o projeto.

Foi feito um site institucional de uma empresa fictícia do ramo
de impressão de embalagens, chamada "Embalagreen". A página inicial
é a "index" e a partir dela se pode navegar para as outras demais.
